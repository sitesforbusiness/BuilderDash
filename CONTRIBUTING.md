# Contributing

This is a public documentation repository for [BuilderDash](https://builderdash.app) — construction project management software for builders, tradespeople, and fit-out contractors.

## What's Here

This repository contains documentation, guides, and feature information for BuilderDash. It's intended to help contractors and builders understand how to use the platform and whether it's the right fit for their business.

## Improving the Docs

If you find errors, outdated information, or gaps in the documentation, feel free to open an issue or submit a pull request.

## About BuilderDash

BuilderDash is built for real construction businesses — builders, fit-out contractors, and trades who need practical software to manage quotes, jobs, invoices, and their team.
